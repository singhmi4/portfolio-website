React Work Order Viewer

Single Page App API

Live Demo Source Code

This is a single page app which shows a list of work orders which can be sorted by date and filtered through.

Work Orders were fetched from an API using axios which were then loaded into a functional component displaying them as cards. FuseJS was utilized to search through the work orders. The state of the shown work orders, the search query, and the sorting feature are kept within the main App file and passed as props to each component.

The app was designed in Material-UI to provide a clean and responsive look to all of the site elements. Eventually, I'll style the compnents with my own CSS library to reduce the need for additional dependencies.

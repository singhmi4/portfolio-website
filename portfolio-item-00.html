Appointment Booking

Full-Stack Web App with Google Calendar Integration

Source Code

This is an appointment booking app with the ability to add events to google calendar.

This app utilizes Google Oauth2 to sign up and login users as well as accessing the user's availability through their Google Calendar API.

The front-end was designed with React because of the dynamic state of the app in order to render data specific to the user.

The back-end is created with Node, Express, MongoDB, and Passport Oauth2 to create and modify user data, appointments and other important data to persist between sessions.

This app is still in development

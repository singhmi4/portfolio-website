Gatsby Fake E-Commerce Website (WIP)

Live Demo Source Code

This is a fake e-commerce website made with the Gatsby React framework.

The Gatsby React framework was used to build the pages and layout components along with a template page for each product. Images were fetched from a headless CMS called Contentful and inserted into the pages while ensuring they are responsive to viewport changes using Graphql.

The app was designed in React Bootstrap, but will be redone without a CSS framework once all of the main features have been implemented.
